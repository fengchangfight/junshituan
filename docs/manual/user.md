# 用户手册

## 基本流程

```
1. 打开 http://localhost:3000 → 进入军师大厅
2. 浏览军师卡片，点击选中（可选 1-5 位）
3. 点击底部"咨询军师"
4. 进入群聊会议室
5. 输入问题 → 军师依次回答
6. 可以继续追问，直到预算耗尽
```

## 军师大厅

- **分类筛选**：按军事家/哲学家/政治家/企业家等分类浏览
- **选中军师**：点击卡片选中，最多选 5 位
- **底部操作栏**：显示已选军师 + "咨询军师"按钮

卡片信息：
```
┌─────────────────────┐
│     [头像占位]       │
│                     │
│     诸 葛 亮        │
│   三国 · 军事家     │
│ 谨慎周密，以史为鉴  │
└─────────────────────┘
```

## 群聊会议室

### 界面布局

```
┌──────────────────────────┐
│ ← 诸葛亮、孙子、王阳明...  │  群聊标题栏
│      3位军师在线          │  可展开查看成员
├──────────────────────────┤
│ [预算条] ¥4.2 / ¥15     │  预算进度 (颜色变化)
├──────────────────────────┤
│                          │
│  系统: 诸葛亮、孙子...   │  系统消息 (居中)
│                          │
│  诸葛亮:                │
│  ┌──────────────────┐   │
│  │ 此事亮已思之...  │   │  军师消息 (左对齐，带头像)
│  └──────────────────┘   │
│                          │
│               ┌───────┐ │
│               │ 用户  │ │  用户消息 (右对齐)
│               └───────┘ │
│                          │
├──────────────────────────┤
│ [向军师团提问...]  [→]  │  输入栏
└──────────────────────────┘
```

### 消息类型

| 类型 | 样式 | 说明 |
|------|------|------|
| 系统消息 | 居中 + 灰色圆角 | 欢迎/提示/预算更新 |
| 军师消息 | 左对齐 + 头像 + 昵称 | 带"思考中..."闪烁动画 |
| 用户消息 | 右对齐 + 渐变气泡 | 无头像 |

### 思考动画

- 军师头像：呼吸光晕
- 消息气泡：三个圆点脉冲 (···)
- 成员面板：对应军师旁脉冲圆点
- 标题栏："某某正在思考..."

## 预算管理

每个议事厅有 ¥15 元预算上限。

- 提问前系统预估费用，超出则拒绝
- 顶部进度条实时显示消费进度
- 预算耗尽后需开启新的议事厅

## 会话恢复

- 每个议事厅有唯一 URL，可以书签保存
- 重新进入后自动恢复全部历史消息
- 军师继续之前的对话上下文回答问题

## 移动端使用

- 完全适配手机屏幕
- 支持微信内置浏览器
- 底部输入栏适配 safe-area（iPhone 全面屏）
- 触摸友好的按钮尺寸 (44px+)
